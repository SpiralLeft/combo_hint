# ComboHint 模组 (Slay the Spire 2 EA)

这是一个用于提示手牌中特殊效果的 **《杀戮尖塔2》EA版** 模组。  
它能自动检测手牌是否包含特定效果，并利用游戏内置对话气泡以及右上角的提示框进行实时提醒。

> **注意：** 这是一个 **vibecoding** 项目，目前仍处于测试阶段。请谨慎使用，**务必及时备份存档**，欢迎反馈BUG。

## 功能概述

- 根据卡牌描述或名称，自动匹配配置文件中的触发词组，命中时会在**弹出对话气泡**，显示“我有XX（效果）”。
- 预配置了多个触发组，每组可在配置中自定义不同的颜色，可自行修改触发组。
- 在游戏中显示**带颜色的效果**，支持多条匹配结果并以“、”号分隔显示。
- **右上角提示框**长期显示用户名和对应的效果。

## 特点

- **动态配置触发组**：无需修改源码，直接编辑combohint.config.json即可调整触发词与颜色，支持高度自定义。
- **防抖合并弹窗**：对抽牌逻辑进行了防抖处理。在连续抽牌结束后，系统会统一检测手牌并只弹出一次提示，避免刷屏。
- **游戏内设置**：可直接在游戏内的设置-游戏设置的底部启用单人连携，启用气泡提示，启用右上提示框。
- **可配置时长**：气泡显示时长可通过 `bubbleDurationSeconds` 灵活调整。
- **融入原版游戏**：直接使用原版游戏的对话气泡和文本框，右上提示框为半透明，悬停时变为不透明。

## 目前可能存在的 BUG

- **其他手牌添加手段**：药水和印牌的情况还不能稳定触发气泡弹出。
- **UI 溢出**：未测试过文本过长时是否会溢出气泡以及提示框。
- **特殊事件兼容性**：遇到特殊效果（如“瓦库接管”）时可能会发生未知情况。
- **防抖失效**：当能力触发极高频的连续抽牌时，防抖窗口可能因过短而失效。

## 之后的修改

- 直接使用卡牌的参数作为触发组，不再使用文本匹配
- 多人模式增加特殊状态连携（中毒/灾厄）
- 快捷键绑定提示框显示
