# ComboHint 模组 (Slay the Spire 2 EA)

这是一个用于提示手牌中特殊效果的 **《杀戮尖塔2》EA版** 模组。  
它能自动检测手牌是否包含特定效果，并利用游戏内置对话气泡进行实时提醒。

## 功能概述
- 根据卡牌描述或名称，自动匹配配置文件中的触发词组。
- 预配置了五类触发组，每组可在配置中自定义不同的颜色，可自行修改触发组。
- 在游戏中显示带颜色的弹泡文本，支持多条匹配结果并以“、”号分隔显示。

## 特点
- **动态配置触发组**：无需修改源码，直接编辑配置文件即可调整触发词与颜色，支持高度自定义。
- **防抖合并弹窗**：对抽牌逻辑进行了防抖处理。在连续抽牌结束后，系统会统一检测手牌并只弹出一次提示，避免刷屏。
- **可配置时长**：气泡显示时长可通过 `bubbleDurationSeconds` 灵活调整（默认为 1.8 秒）。
- **单人模式开关**：可通过 `enableSinglePlayerHint` 自由配置是否在单人模式下启用提示。


## 目前可能存在的 BUG
* **复杂抽牌逻辑**：在某些极端复杂的抽牌情况下，命中可能不完整。
* **特殊事件兼容性**：遇到特殊效果（如“瓦库接管”）时可能会发生未知情况。
* **防抖失效**：当能力触发极高频的连续抽牌时，防抖窗口可能因过短而失效。
* **UI 溢出**：若同时命中的效果过多，文本可能会超出对话气泡的显示范围。
